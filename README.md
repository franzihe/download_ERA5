# Regrid global data


## Variables


For my study:
- In pressure levels:
  - _t   : temperature_
  - _clwc : specific_cloud_liquid_water_content_
  - _ciwc : specific_cloud_ice_water_content_
  - _cswc : specific_snow_water_content_
- In single levels:
  - _2t   : 2m_temperature_
  - _msr  : mean_snowfall_rate_
  - _sf   : snowfall_
